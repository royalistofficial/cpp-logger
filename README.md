# Многопоточная запись сообщений в журнал (C++17)

Динамическая библиотека логирования + консольное многопоточное
приложение, демонстрирующее её работу.

**Библиотека** пишет текстовые сообщения в файл журнала. При инициализации
принимает имя файла и уровень важности по умолчанию; сообщения с уровнем ниже
заданного отбрасываются. В журнал попадают текст, уровень и время получения
сообщения. Уровень по умолчанию можно менять после инициализации.

**Приложение** читает из консоли сообщение и (необязательно) его уровень,
потокобезопасно передаёт данные в отдельный поток записи и сразу возвращается к
приёму нового ввода. Параметры запуска — имя файла журнала и уровень по умолчанию.

Ограничения: только STL, стандарт C++17, сборка через `make` компилятором `g++`,
целевая ОС — актуальные Ubuntu/Debian.


Формат ввода: `<уровень> <текст>` либо просто `<текст>` — тогда берётся уровень
по умолчанию. Пустая строка завершает работу.

## Структура

```
include/logger/     публичный API библиотеки
src/logger/         приватная реализация
app/                консольное приложение
tests/              тесты
Makefile
```
